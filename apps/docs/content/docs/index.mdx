# Turbo Modern Starter Documentation

Welcome to the Turbo Modern Starter documentation! This monorepo template includes everything you need to build modern web applications and browser extensions.

## Features

- Turborepo monorepo setup
- Next.js web application
- Browser extension with Vite
- Shared UI components
- Documentation site
- End-to-end testing with Playwright
- Unit testing with Vitest
- GitHub Actions CI/CD workflows
