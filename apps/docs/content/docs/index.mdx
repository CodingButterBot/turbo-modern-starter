# Turbo Modern Starter

Welcome to the official documentation for Turbo Modern Starter, a modern and complete monorepo starter kit for building full-stack applications.

## What is Turbo Modern Starter?

Turbo Modern Starter is a comprehensive starter template that combines the power of:

- TypeScript for type-safe code
- Next.js for web applications
- Tailwind CSS for styling
- Turborepo for monorepo management
- Vitest for unit and integration testing
- Playwright for E2E testing
- Directus for headless CMS
- GitHub Actions for CI/CD

This documentation will help you get up and running with the template and explore its features in depth.
